자바스크립트 언어를 간편하게 사용할 수 있도록 단순화시킨
        오픈 소스 기반의 자바스크립트 라이브러리이다. 

        사용 방법
            1. 다운로드: 제이쿼리 홈페이지에서 다운로드
                제이쿼리 다운로드 하기
                
                

            2. CDN(주소참조활용): Content Delivery Network, 네트워크(인터넷) 에서 컨텐트를 가져오겠다는 뜻이다. 
                
                

        [기본 문법]
            1. id 문법
                $(document).ready(function() { // 문서에서, 어떠한 행동을 할 준비가 되어있다는 의미의 코드이다. 
                    $("#id이름").click(function() { // id이름이 지정된 태그를 클릭했을 때 실행할 동작
                        동작에 관련된 코드 작성
                    })
                })

            2. 이벤트리스너 문법 (onclick, keyon, onchange, ...)
                function 이벤트리스너에지정한이름() {
                    $("id이름").기능작성();
                }

        [선택자 문법]
            1. .css() 선택
                $("태그 이름 또는 #id이름 또는 .class이름").css("배경", "색상");

            2. .toggle() 선택
                $("태그 이름 또는 #id이름 또는 .class이름").toggle("속도조절"); // "속도조절" 은 생략 가능하다. 

            3. .slideToggle() 선택: 토글과 비슷하지만, 주로 이미지를 보여주거나 숨길 때 사용한다. 

            4. .hide() .show() 선택

            5. .append() 선택: 태그나 값을 추가할 때 사용한다.
